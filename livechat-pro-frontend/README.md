# LiveChat Pro Frontend

Premium React + Vite + Tailwind dashboard for the LiveChat Pro backend.

## Setup

```bash
cp .env.example .env
npm install
npm run dev
```

## Backend endpoints
- API base: `VITE_API_URL=http://localhost:5000/api/v1`
- Socket server: `VITE_SOCKET_URL=http://localhost:5000`
